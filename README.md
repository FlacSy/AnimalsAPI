# Random Animal API

## Введение
Это документация описывает API для получения случайного изображения животного из заданной категории. API предоставляет возможность запросить случайное изображение животного, указав желаемую категорию животного. Если категория не указана, будет возвращено случайное изображение из списка поддерживаемых категорий.

## Описание эндпоинтов

### 1. `/random_image`

#### Метод
- **GET**

#### Параметры запроса
- **animal** (строка, необязательный): Категория животного. Если не указана, будет выбрано случайное животное из списка поддерживаемых категорий.

#### Ответ
```json
{
    "image_url": "http://127.0.0.1:5000/animals/{animal}/{filename}.jpg",
    "request_type": "GET",
    "requested_animal": "{animal}",
    "status_code": 200
}
```

#### Ошибки
- **400 Bad Request**: В случае, если параметр `animal` указан неверно или не существует. В ответе будет предоставлено сообщение об ошибке и список поддерживаемых параметров.

### 2. `/animals/<animal>/<image>`

#### Метод
- **GET**

#### Описание
Этот эндпоинт предназначен для получения изображения животного по указанному пути.

## Запуск приложения
Для запуска приложения выполните следующую команду:
```bash
python app.py
```

## Настройка
Для работы приложения требуется наличие конфигурационного файла `config.py`, содержащего путь к папке с изображениями животных и список поддерживаемых категорий животных. 

### Пример файла `config.py`
```python
{
    'path': {
        'animals': '/путь/к/папке/с/папками/изображений/'
    },
    'animals': {
        'support': ['категория_1', 'категория_2', ...]
    }
}
```
